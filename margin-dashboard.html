数据口径：本仪表盘为全市场（沪市+深市+北证）融资融券汇总，每交易日一行。所参照的样例站 margin-dashboard-4z2.pages.dev 为上交所（沪市）单独数据（如 2026-07-20 沪市两融≈1.37万亿），本版用同一数据源（东方财富）做了更完整的全市场口径，数值约为其 2 倍，属正常差异。

实时更新机制：页面打开即向东方财富接口直连拉取最新数据（浏览器跨域，Access-Control-Allow-Origin:*），每 10 分钟自动刷新；若接口临时不可用，则回退到仓库内的 margin-data.json 缓存快照。另有 GitHub Actions 每日收盘后自动刷新该缓存。

实现要点（模仿原站）：Cloudflare Pages 版用 Pages Functions/Worker 在服务端定时抓取上交所数据并缓存；本静态版用纯前端直连 + 定时刷新 + JSON 缓存兜底，无需服务器即可部署到 GitHub Pages / Gitee Pages。颜色遵循 A 股惯例：红涨 / 绿跌。

⚠️ 本页为公开数据可视化，所有数值以交易所/数据商原始披露为准；不构成任何投资建议。两融余额是杠杆情绪的高频指标，但需结合价格、成交、估值综合判断。
