# 设计文档：把知识库改造成个人博客「财务自由之路」

日期：2026-07-22
状态：已实现并部署

## 1. 目标与范围

把现有「投资理财-复利与财务自由」知识库（13 篇文章 + 3 个实时仪表盘）包装成一个**个人博客**，命名为「财务自由之路」。

用户选定的范围（轻量改造）：
- 重做首页为博客形态（品牌名、关于区块、文章分类、保留实时工具）。
- 13 篇文章页与 3 个仪表盘**沿用现有样式**，通过统一导航/页脚串联。
- 不逐一改写 13 个文章文件的内部结构（避免高改动风险）。

## 2. 架构与改动

### 2.1 首页 `index.html`（重写为博客首页）
- 品牌名统一为「财务自由之路」；更新 `<title>` / `<meta description>`。
- 顶部 sticky 导航：关于 / 文章系列 / 实时工具 / 美国通胀·利率 / 国家队护盘。
- Hero：博客名 + 个人语气简介 + 数据指标（13篇 / 4层 / 20+计算器 / 3工具）。
- 新增「关于这个博客」区块：说明这是写给自己的投资理财说明书、四层递进结构、免责声明。
- 文章系列：保留「认知 → 数学 → 落地 → 目标」四层卡片地图（复用 13 篇）。
- 实时数据工具：保留 3 个仪表盘入口。
- 页脚：统一免责声明 + 「财务自由之路 · 个人博客」。

### 2.2 共享串联 `assets/site.js`（增强，所有子页面已加载）
原逻辑：非首页注入「📑 目录」悬浮按钮返回 index.html。
增强为：
- 悬浮按钮改名为「🏠 财务自由之路」（返回博客首页）。
- 自动为**浅色主题且无 footer** 的子页面补一个统一博客页脚（免责 + 返回首页）。
- 深色调仪表盘页（如 national-team）跳过自动页脚，避免配色冲突；仅靠悬浮按钮串联。

### 2.3 修复遗漏
- `national-team.html` 此前未引用 `assets/site.js`，已补 `<script src="assets/site.js">`，使护盘图谱页也获得返回博客按钮。

## 3. 关键决策与取舍

- **复用 site.js 而非新建脚本 / 不改 13 个文件**：因 site.js 已被全部页面加载，只增强一处即可串联全部子页，风险最低。
- **不强行给深色仪表盘注入浅色页脚**：避免配色冲突，靠悬浮按钮保证可达性。
- **保留知识库四层结构**：既是内容骨架，也是博客的「系列/专栏」分类，无需重新组织文章。

## 4. 验证

- `node --check assets/site.js` 通过。
- 线上部署后：`index.html`、`assets/site.js` 及各数据脚本均返回 200。
- 子页面右下角出现「🏠 财务自由之路」悬浮按钮，点击返回博客首页。
