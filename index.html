部署到 wuheping.top
本站为纯静态页面（无构建步骤），已包含 CNAME、.nojekyll 与部署说明，可直接推送到 GitHub / Gitee Pages。

GitHub Pages

  • 新建仓库 → 添加远程 git remote add github <url> → git push -u github main
  • Settings → Pages → Source 选 main 分支、/ (root)
  • Custom domain 填 wuheping.top，DNS 添加 4 条 A 记录指向 185.199.108.153 / .109 / .110 / .111，开启 HTTPS

Gitee Pages

  • 新建公开仓库 → git remote add gitee <url> → git push -u gitee master
  • 服务 → Gitee Pages → 根目录部署；自定义域名需 Gitee Pages Pro

双端同步

  • git push github main && git push gitee master
  • 完整步骤见仓库内 README.md
